# 黑马点评 HM-Dianping
基于 SpringBoot + Redis 实战的本地生活平台项目，复刻大众点评业务，主打**Redis全场景应用、高并发缓存优化、优惠券秒杀、分布式锁**，是Java后端经典实战项目。

## 项目简介
黑马点评是前后端分离的本地生活服务平台，对标大众点评。
项目围绕**Redis**进行全业务场景实战，覆盖缓存优化、分布式锁、限流、签到统计、关注点赞、好友关注、探店笔记、高并发优惠券秒杀等核心业务，解决缓存穿透、击穿、雪崩、秒杀超卖、一人一单等企业级问题。

## 技术栈
### 后端
- SpringBoot 2.7.x
- MyBatis-Plus
- MySQL 8.0
- Redis 6.x
- JWT 登录令牌
- Maven
- Tomcat

### 前端
- Vue + ElementUI
- Axios

### 核心中间件 & 技术亮点
- Redis 多种数据结构实战（String/Hash/Set/ZSet/List）
- 缓存三大问题解决方案：**缓存穿透、缓存击穿、缓存雪崩**
- Cache Aside 缓存旁路模式、缓存一致性
- 分布式锁（Redisson、Redis+Lua）
- 优惠券秒杀、防超卖、一人一单、接口限流
- 用户签到、连续签到、UV统计
- 好友关注、共同关注、笔记点赞、分页查询

## 项目功能
1. **短信登录**：手机号验证码登录、JWT令牌登录校验、拦截器权限控制
2. **商户查询**：店铺信息浏览、多条件查询、缓存优化
3. **用户签到**：基于Redis BitMap实现签到、连续签到统计
4. **关注功能**：用户关注博主、粉丝列表、共同关注
5. **探店笔记**：笔记发布、浏览、点赞、排行榜
6. **优惠券秒杀**：高并发秒杀、分布式锁、防超卖、一人一单
7. **附近商户**：GEO地理位置实现附近商家搜索
